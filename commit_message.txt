fix: Corriger les bugs de redirection et d'affichage des cartes du pipeline

Ce commit résout deux problèmes majeurs :
- Le bug de redirection : Le ProtectedRoute attend désormais que l'AuthContext ait terminé de vérifier l'état de connexion, évitant ainsi les redirections prématurées.
- Le problème d'affichage des cartes du pipeline : La couleur du texte des cartes dans le pipeline est forcée à être sombre pour assurer une meilleure lisibilité sur les fonds clairs.
